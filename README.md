# CampusConnect

CampusConnect is a web-based student collaboration platform built using Java.  
The system helps university students connect with each other to share academic services such as tutoring, study notes, books, project collaboration, and programming help.

## Project Idea
The main goal of this project is to create a simple and useful university platform where students can:
- create accounts
- post services or requests
- browse available posts
- communicate through the platform
- manage their own content

This project is being developed in two stages:
1. **Java Servlets + JDBC + MySQL + HTML/CSS/JavaScript**
2. **Spring Boot version** as an advanced upgrade

## Technologies Used

## Features
### User Features
- User registration and login
- Profile management
- Create, edit, and delete posts
- Browse services and requests
- Search by category
- View post details
- Contact or respond to other users

### Admin Features
- Manage users
- Manage posts
- Remove inappropriate content
- Monitor platform activity

## Categories

## Database Concepts
The system uses a relational database to manage:
- users
- posts
- categories
- requests/responses
- admin controls
