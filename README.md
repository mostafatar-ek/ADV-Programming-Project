# CampusConnect

CampusConnect is a student web platform built using Java Servlets and JDBC.  
It allows students to register, create posts, browse academic services, and connect with others for tutoring, study materials, books, and project collaboration.

## Technologies
- Java
- Servlets
- JDBC
- MySQL
- HTML/CSS/JavaScript
- Apache Tomcat

## Main Features
- User registration and login
- Create and manage posts
- Browse and search services
- Category filtering
- Admin dashboard
- Database integration

## Future Work
The project will later be upgraded using Spring Boot to improve scalability, structure, and modern backend development practices.
